<h1>СКАН-сервис</h1>
<p>Сервис для поиска поиска публикаций о компаниях по ИНН.</p>

<h2>Компоненты: </h2>
<ol>
  <li>Главная страницы</li>
  <li>Форма авторизации</li>
  <li>Форма параметров запроса</li>
  <li>Страница вывода результата</li>
</ol>

<h2>Команды для запуска: </h2>
<ul>
  <li>npm install</li>
  <li>npm run build</li>
  <li>npm start</li>
</ul>
